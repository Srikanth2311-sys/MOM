
## ** Prompt Framework Examples (Short Version)**

### **AIM**

```
Write a summary (Action) of this document (Input) in 5 bullet points (Mode).
```

### **CRISP**

```
Summarize this document (Input) in 5 key points (Steps) for a business review (Purpose),
while acting as a strategy consultant (Role), with the context that this is for a
leadership meeting (Context).
```

### **ROLE**

```
Act as an Angular expert (Role) and explain RxJS Subjects (Objective)
in 5 bullet points (Limitations) with real project use cases (Expectations).
```

### **COSTAR**

```
Explain this document (Objective) using a simple style and professional tone (Style & Tone)
for a business audience (Audience), and return the output as 5 bullet points (Response Format).
```
